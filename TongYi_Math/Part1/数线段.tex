Example:一条直线上有$n$个点，则共有多少条线段？
\clearpage
Hints:
\begin{itemize}
\item 记线段条数为$f(n)$, 计算$f(1), f(2), f(3), f(4), f(5)$
\item 根据上面的数据，猜测$f(k)$与$f(k+1)$的关系，根据关系给出逻辑推理
\item 根据$f(k), f(k+1)$之间的关系求出$f(n)$的表达式
\item 利用所得$f(n)$表达式，检验$f(1), \cdots, f(5)$的值，并计算$f(100)$
\end{itemize}
